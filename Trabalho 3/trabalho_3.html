Trabalho 3 - Desenvolvimento do Sistema de Processamento Visual (SPV)

Descrição Funcional

O presente código realiza a detecção e reconhecimento de placas veiculares a partir de imagens capturadas por uma webcam, utilizando técnicas de processamento digital de imagens e OCR (Reconhecimento Óptico de Caracteres). O programa aplica uma série de etapas para processar a imagem, identificar possíveis regiões contendo placas, e extrair texto válido dessas regiões. A seguir, é detalhado o funcionamento do código e o motivo da utilização de cada técnica implementada.

Captura de Imagem

O código utiliza a biblioteca OpenCV para capturar frames de uma webcam em tempo real. Cada frame é processado em busca de possíveis placas veiculares. Esta abordagem em tempo real é adequada para sistemas de monitoramento e controle de acesso.

Pré-processamento da Imagem

Após capturar um frame, várias técnicas de pré-processamento são aplicadas para melhorar a qualidade da imagem e facilitar a identificação de regiões relevantes:

  • Conversão para Escala de Cinza - A imagem é convertida para tons de cinza, eliminando a influência de cores e reduzindo a complexidade computacional, pois as operações passam a ser realizadas em um único canal de intensidade.

  • Redimensionamento - A imagem é redimensionada com um fator de escala (1.2) utilizando interpolação cúbica, para melhorar a resolução das características e aumentar a precisão do OCR.

  • Dilatação e Erosão - Essas operações morfológicas são aplicadas para destacar os contornos das regiões de interesse. A dilatação expande as áreas mais brilhantes, enquanto a erosão reduz áreas menores e remove ruídos isolados. Estas operações ajudam a realçar os contornos de caracteres nas placas.

  • Suavização e Limiarização - O filtro Gaussiano e o filtro bilateral são utilizados para reduzir ruídos preservando bordas importantes. Métodos de limiarização global (Otsu) e adaptativa são aplicados para binarizar a imagem, convertendo-a em preto e branco. Isso facilita a segmentação de caracteres no OCR.

Detecção de Regiões Candidatas

Após o pré-processamento, contornos na imagem são detectados usando a função `findContours`. Cada contorno é envolvido por um retângulo delimitador (bounding box), e critérios específicos de tamanho e proporção são aplicados para filtrar regiões que se assemelhem a placas veiculares. A proporção típica de placas é utilizada como referência (largura maior que altura e proporções específicas entre 2:1 e 5:1).

Processamento das Regiões de Interesse (ROI)

Para cada região candidata identificada:

  • A área correspondente é extraída como uma região de interesse (ROI).

  • A ROI passa por processamento adicional (escala de cinza, redimensionamento, dilatação, erosão, suavização e limiarização).

  • Após o processamento, a ROI é salva como uma imagem em uma pasta local para verificação e armazenamento.

Reconhecimento Óptico de Caracteres (OCR)

A biblioteca Tesseract é utilizada para realizar o OCR na ROI processada. O texto detectado é tratado da seguinte forma:

  • Remoção de caracteres indesejados - Apenas caracteres alfanuméricos são mantidos, e tudo é convertido para maiúsculas.

  • Validação do comprimento - Apenas sequências de exatamente 7 caracteres são consideradas válidas, eliminando leituras incorretas.

Integração com Banco de Dados

As placas reconhecidas são comparadas com um banco de dados local que contém informações sobre motoristas e veículos. Caso a placa seja encontrada no banco de dados, o programa exibe as informações correspondentes (nome do motorista e modelo do carro). Caso contrário, uma mensagem de acesso negado é apresentada.

Salvar Imagens Processadas

O código salva as imagens das regiões candidatas (com o processamento completo aplicado) em uma pasta chamada "placas_salvas". As imagens são nomeadas com base na data e hora da captura para facilitar a organização e o rastreamento.


Justificativa das Técnicas Utilizadas

Processamento Digital de Imagens

As técnicas aplicadas no pré-processamento ajudam a remover ruídos, destacar características relevantes e segmentar os caracteres de maneira mais precisa. Essas etapas são fundamentais para aumentar a confiabilidade do OCR, especialmente em condições adversas, como iluminação variável ou presença de sujeira nas placas.

Detecção de Contornos

A detecção de contornos com filtragem baseada em proporção e tamanho reduz a quantidade de regiões processadas pelo OCR, otimizando o desempenho do sistema e diminuindo a chance de erros.

OCR com Tesseract

O Tesseract é uma biblioteca amplamente utilizada para reconhecimento de texto, oferecendo suporte a múltiplos idiomas e uma integração eficiente com o OpenCV. Ele permite transformar as regiões processadas em texto que pode ser validado e armazenado.


Sistema de Reconhecimento de Placas Veiculares

Antes da execução do código, é necessário executar alguns comandos para configurar o CMake e compilar o código: 



$ cmake .
$ g++ -o placas_opencv placas_opencv.cpp `pkg-config --cflags --libs opencv4` -ltesseract `sdl2-config --cflags --libs` -lSDL2_ttf
$ ./placas_opencv

O código do programa está disposto abaixo: 



/* Grupo VGA - Sistema de Reconhecimento de Placas Veiculares
André Marques da Silva RA: 11202021067
Gabriel Batista Veloso RA: 11201921267
Vinícius de Souza Feitosa RA: 11202021889

#include <opencv2/opencv.hpp>
#include <tesseract/baseapi.h>
#include <iostream>
#include <chrono>
#include <ctime>
#include <sstream>
#include <filesystem>
#include <regex>
#include <map>

using namespace cv;
using namespace std;
namespace fs = std::filesystem;

int main() {
    // Banco de dados de placas cadastradas
    map> bancoDeDados = {
        {"RIO2A18", {"Andre", "Ford Fiesta"}},
        {"ABC1D34", {"Gabriel", "Chevrolet Onix"}},
        {"PBA2O19", {"Vinicius", "Honda Fit"}}
    };

    // Verificar e criar a pasta "placas_salvas" para armazenar capturas de imagens
    string pasta = "placas_salvas";
    if (!fs::exists(pasta)) {
        fs::create_directory(pasta);
        cout << "Pasta 'placas_salvas' criada." << endl;
    }

    // Abrir a câmera (webcam)
    VideoCapture cap(0);
    if (!cap.isOpened()) {
        cerr << "Erro ao abrir a webcam!" << endl;
        return -1;
    }

    // Inicializar o OCR (Tesseract)
    tesseract::TessBaseAPI ocr;
    ocr.Init(NULL, "eng");

    cout << "Pressione a barra de espaço para verificar a placa detectada." << endl;
    cout << "Pressione 'q' para sair." << endl;

    while (true) {
        Mat frame;
        cap >> frame; // Captura um frame da webcam

        if (frame.empty()) {
            cerr << "Erro ao capturar o frame!" << endl;
            break;
        }

        // Mostrar o feed da câmera ao vivo
        imshow("Sistema de reconhecimento de placas veiculares", frame);

        char key = (char)waitKey(30);
        if (key == 'q' || key == 'Q') {
            break;
        } else if (key == ' ') { // Pressionou espaço para capturar
            // Pré-processamento
            Mat gray, processed, roi; // Declarar `roi` fora do loop
            cvtColor(frame, gray, COLOR_BGR2GRAY); // Converter para escala de cinza
            
            // Passo 1: Redimensionamento para melhorar a qualidade de leitura
            resize(gray, gray, Size(), 1.2, 1.2, INTER_CUBIC);

            // Passo 2: Aplicar dilatação e erosão para destacar as características principais
            Mat kernel = Mat::ones(1, 1, CV_8U); // Kernel de 1x1
            dilate(gray, processed, kernel, Point(-1, -1), 1); // Dilatação
            erode(processed, processed, kernel, Point(-1, -1), 1); // Erosão

            // Passo 3: Aplicar suavização e limiarização
            GaussianBlur(processed, processed, Size(5, 5), 0);
            threshold(processed, processed, 0, 255, THRESH_BINARY + THRESH_OTSU);

            // Passo 4: Detectar contornos para identificar regiões candidatas a placas
            vector> contornos;
            findContours(processed, contornos, RETR_EXTERNAL, CHAIN_APPROX_SIMPLE);

            vector candidatos; // Armazenar retângulos candidatos
            for (const auto& contorno : contornos) {
                Rect retangulo = boundingRect(contorno);
                float proporcao = (float)retangulo.width / retangulo.height;

                // Filtrar retângulos com base no tamanho e na proporção (ex.: 4:1 típico de placas)
                if (retangulo.width > 100 && retangulo.height > 25 && proporcao > 2 && proporcao < 5) {
                    candidatos.push_back(retangulo);
                }
            }

            // Passo 5: Iterar sobre as regiões candidatas e aplicar OCR
            string placaDetectada;
            for (const auto& retangulo : candidatos) {
                roi = gray(retangulo); // Extrair região de interesse

                // Aplicar processamento à ROI antes de usar OCR e salvar
                Mat roiProcessada;
                resize(roi, roi, Size(), 1.2, 1.2, INTER_CUBIC); // Melhorar detalhes
                dilate(roi, roiProcessada, kernel, Point(-1, -1), 1); // Dilatação
                erode(roiProcessada, roiProcessada, kernel, Point(-1, -1), 1); // Erosão
                GaussianBlur(roiProcessada, roiProcessada, Size(5, 5), 0);
                threshold(roiProcessada, roiProcessada, 0, 255, THRESH_BINARY + THRESH_OTSU);

                // Aplicar OCR na região processada
                ocr.SetImage(roiProcessada.data, roiProcessada.cols, roiProcessada.rows, 1, roiProcessada.step);
                string texto = string(ocr.GetUTF8Text());

                // Limpar o texto detectado
                texto = regex_replace(texto, regex("[^A-Za-z0-9]"), "");
                transform(texto.begin(), texto.end(), texto.begin(), ::toupper);

                // Validar comprimento e evitar leituras inválidas
                if (texto.size() == 7) {
                    placaDetectada = texto;

                    // Salvar imagem processada da ROI
                    auto now = chrono::system_clock::now();
                    time_t now_c = chrono::system_clock::to_time_t(now);
                    stringstream datetime;
                    datetime << put_time(localtime(&now_c), "%H-%M-%S_%d-%m-%Y");

                    string filename = pasta + "/placa_" + datetime.str() + ".jpg";
                    imwrite(filename, roiProcessada); // Salvar imagem processada
                    cout << "Imagem salva como: " << filename << endl;
                    break; // Usar a primeira leitura válida encontrada
                }
            }

            // Verificar se uma placa válida foi detectada
            if (!placaDetectada.empty()) {
                cout << "Placa detectada: " << placaDetectada << endl;

                // Verificar no banco de dados
                if (bancoDeDados.find(placaDetectada) != bancoDeDados.end()) {
                    cout << "Acesso permitido!" << endl;
                    cout << "Motorista: " << bancoDeDados[placaDetectada].first << endl;
                    cout << "Modelo do carro: " << bancoDeDados[placaDetectada].second << endl;
                } else {
                    cout << "Acesso negado! Placa não encontrada no sistema." << endl;
                }
            } else {
                cout << "Nenhuma placa válida detectada." << endl;
            }
        }
    }

    // Finalizar OCR e liberar recursos
    ocr.End();
    cap.release();
    destroyAllWindows();

    cout << "Programa encerrado." << endl;
    return 0;
}
