Trabalho 3 - Desenvolvimento do Sistema de Processamento Visual (SPV)

Descrição Funcional

Código preliminar

Entrada: Vídeo em tempo real capturado pela webcam. O presente código realiza a detecção e reconhecimento de placas veiculares a partir de imagens capturadas por uma webcam, utilizando técnicas de processamento digital de imagens e OCR (Reconhecimento Óptico de Caracteres). O programa aplica uma série de etapas para processar a imagem, identificar possíveis regiões contendo placas, e extrair texto válido dessas regiões. A seguir, é detalhado o funcionamento do código e o motivo da utilização de cada técnica implementada. ### 1. Captura de Imagem O código utiliza a biblioteca OpenCV para capturar frames de uma webcam em tempo real. Cada frame é processado em busca de possíveis placas veiculares. Esta abordagem em tempo real é adequada para sistemas de monitoramento e controle de acesso. ### 2. Pré-processamento da Imagem Após capturar um frame, várias técnicas de pré-processamento são aplicadas para melhorar a qualidade da imagem e facilitar a identificação de regiões relevantes: - **Conversão para Escala de Cinza:** A imagem é convertida para tons de cinza, eliminando a influência de cores e reduzindo a complexidade computacional, pois as operações passam a ser realizadas em um único canal de intensidade. - **Redimensionamento:** A imagem é redimensionada com um fator de escala (1.2) utilizando interpolação cúbica, para melhorar a resolução das características e aumentar a precisão do OCR. - **Dilatação e Erosão:** Essas operações morfológicas são aplicadas para destacar os contornos das regiões de interesse. A dilatação expande as áreas mais brilhantes, enquanto a erosão reduz áreas menores e remove ruídos isolados. Estas operações ajudam a realçar os contornos de caracteres nas placas. - **Suavização e Limiarização:** - O filtro Gaussiano e o filtro bilateral são utilizados para reduzir ruídos preservando bordas importantes. - Métodos de limiarização global (Otsu) e adaptativa são aplicados para binarizar a imagem, convertendo-a em preto e branco. Isso facilita a segmentação de caracteres no OCR. ### 3. Detecção de Regiões Candidatas Após o pré-processamento, contornos na imagem são detectados usando a função `findContours`. Cada contorno é envolvido por um retângulo delimitador (bounding box), e critérios específicos de tamanho e proporção são aplicados para filtrar regiões que se assemelhem a placas veiculares. A proporção típica de placas é utilizada como referência (largura maior que altura e proporções específicas entre 2:1 e 5:1). ### 4. Processamento das Regiões de Interesse (ROI) Para cada região candidata identificada: - A área correspondente é extraída como uma região de interesse (ROI). - A ROI passa por processamento adicional (escala de cinza, redimensionamento, dilatação, erosão, suavização e limiarização). - Após o processamento, a ROI é salva como uma imagem em uma pasta local para verificação e armazenamento. ### 5. Reconhecimento Óptico de Caracteres (OCR) A biblioteca Tesseract é utilizada para realizar o OCR na ROI processada. O texto detectado é tratado da seguinte forma: - **Remoção de caracteres indesejados:** Apenas caracteres alfanuméricos são mantidos, e tudo é convertido para maiúsculas. - **Validação do comprimento:** Apenas sequências de exatamente 7 caracteres são consideradas válidas, eliminando leituras incorretas. - **Verificação de padrões:** Leituras contendo a substring "BR" são descartadas, pois não correspondem a padrões de placas. ### 6. Integração com Banco de Dados As placas reconhecidas são comparadas com um banco de dados local que contém informações sobre motoristas e veículos. Caso a placa seja encontrada no banco de dados, o programa exibe as informações correspondentes (nome do motorista e modelo do carro). Caso contrário, uma mensagem de acesso negado é apresentada. ### 7. Salvar Imagens Processadas O código salva as imagens das regiões candidatas (com o processamento completo aplicado) em uma pasta chamada "placas_salvas". As imagens são nomeadas com base na data e hora da captura para facilitar a organização e o rastreamento. ### Justificativa das Técnicas Utilizadas 1. **Processamento Digital de Imagens:** As técnicas aplicadas no pré-processamento ajudam a remover ruídos, destacar características relevantes e segmentar os caracteres de maneira mais precisa. Essas etapas são fundamentais para aumentar a confiabilidade do OCR, especialmente em condições adversas, como iluminação variável ou presença de sujeira nas placas. 2. **Detecção de Contornos:** A detecção de contornos com filtragem baseada em proporção e tamanho reduz a quantidade de regiões processadas pelo OCR, otimizando o desempenho do sistema e diminuindo a chance de erros. 3. **OCR com Tesseract:** O Tesseract é uma biblioteca amplamente utilizada para reconhecimento de texto, oferecendo suporte a múltiplos idiomas e uma integração eficiente com o OpenCV. Ele permite transformar as regiões processadas em texto que pode ser validado e armazenado. 4. **Validação de Texto:** A filtragem baseada em comprimento e formato das placas é essencial para evitar leituras incorretas e assegurar que apenas sequências válidas sejam consideradas. Este passo reduz significativamente falsos positivos. 5. **Armazenamento de Imagens:** Salvar as imagens processadas permite realizar verificações manuais e auditorias posteriores, além de criar um histórico das placas detectadas. ### Considerações Finais O sistema foi projetado para ser robusto e eficiente na leitura de placas veiculares em tempo real, utilizando uma combinação de técnicas de processamento de imagens e OCR. Ele pode ser aplicado em sistemas de controle de acesso, vigilância e monitoramento de tráfego. O uso das técnicas mencionadas garante maior precisão na detecção e reconhecimento, mesmo em cenários desafiadores.

using namespace cv;
using namespace std;
namespace fs = std::filesystem;

int main() {
    // Banco de dados de placas cadastradas (simulado com um map)
    map> bancoDeDados = {
        {"BRA2E19", {"Andre", "Ford Fiesta"}},
    };

    // Verificar e criar a pasta "placas_salvas" se não existir
    string pasta = "placas_salvas";
    if (!fs::exists(pasta)) {
        fs::create_directory(pasta);
        cout << "Pasta 'placas_salvas' criada." << endl;
    }

    // Abrir a câmera (webcam)
    VideoCapture cap(0); // Usar a câmera padrão
    if (!cap.isOpened()) {
        cerr << "Erro ao abrir a webcam!" << endl;
        return -1;
    }

    // Inicializar o OCR (Tesseract)
    tesseract::TessBaseAPI ocr;
    ocr.Init(NULL, "eng"); // Inicializa o OCR com o idioma inglês

    cout << "Pressione a barra de espaço para verificar a placa detectada." << endl;
    cout << "Pressione 'q' para sair." << endl;

    while (true) {
        Mat frame;
        cap >> frame; // Captura um frame da webcam

        if (frame.empty()) {
            cerr << "Erro ao capturar o frame!" << endl;
            break;
        }

        // Mostrar o feed da câmera ao vivo
        imshow("Sistema de reconhecimento de placas veiculares", frame);

        char key = (char)waitKey(30);
        if (key == 'q' || key == 'Q') {
            break; // Sair do loop se 'q' for pressionado
        } else if (key == ' ') { // Se a barra de espaço for pressionada
            // Pré-processamento da imagem antes de fazer o OCR
            Mat gray, processed;

            // Converter para escala de cinza
            cvtColor(frame, gray, COLOR_BGR2GRAY);

            // Redimensionar a imagem para aumentar a qualidade
            resize(gray, processed, Size(), 1.2, 1.2, INTER_CUBIC);

            // Aplicar operações morfológicas para reduzir ruídos
            Mat kernel = getStructuringElement(MORPH_RECT, Size(1, 1));
            dilate(processed, processed, kernel, Point(-1, -1), 1);
            erode(processed, processed, kernel, Point(-1, -1), 1);

            // Aplicar suavização e limiarização
            GaussianBlur(processed, processed, Size(5, 5), 0);
            threshold(processed, processed, 0, 255, THRESH_BINARY + THRESH_OTSU);

            // Configurar o Tesseract para usar a imagem processada
            ocr.SetImage(processed.data, processed.cols, processed.rows, 1, processed.step);

            // Realizar o OCR e obter o texto
            string placaTexto = string(ocr.GetUTF8Text());

            // Limpar a string do texto da placa
            placaTexto = regex_replace(placaTexto, regex("[^A-Za-z0-9]"), ""); // Remove caracteres especiais e hifens
            transform(placaTexto.begin(), placaTexto.end(), placaTexto.begin(), ::toupper); // Converte para maiúsculas

            if (!placaTexto.empty()) {
                cout << "Texto detectado (normalizado): " << placaTexto << endl;

                // Verificar se a placa está no banco de dados
                if (bancoDeDados.find(placaTexto) != bancoDeDados.end()) {
                    cout << "Acesso permitido!" << endl;
                    cout << "Motorista: " << bancoDeDados[placaTexto].first << endl;
                    cout << "Modelo do carro: " << bancoDeDados[placaTexto].second << endl;
                } else {
                    cout << "Acesso negado! Placa não encontrada no sistema." << endl;
                }

                // Salvar a imagem da placa (opcional)
                auto now = chrono::system_clock::now();
                time_t now_c = chrono::system_clock::to_time_t(now);
                stringstream datetime;
                datetime << put_time(localtime(&now_c), "%H-%M-%S_%d-%m-%Y");
                
                // Adicionar a data/hora na imagem
                stringstream datetimeImage;
                datetimeImage << put_time(localtime(&now_c), "%H:%M:%S %d/%m/%Y");
                putText(processed, datetimeImage.str(), Point(10, 50), FONT_HERSHEY_SIMPLEX, 1.0, Scalar(255, 255, 255), 2);
                

                string filename = pasta + "/placa_" + datetime.str() + ".jpg";
                if (!imwrite(filename, processed)) {
                    cerr << "Erro ao salvar a imagem!" << endl;
                } else {
                    cout << "Imagem salva como: " << filename << endl;
                }
            } else {
                cout << "Nenhuma placa detectada." << endl;
            }
        }
    }

    // Finalizar o OCR e liberar recursos
    ocr.End();
    cap.release();
    destroyAllWindows();

    cout << "Programa encerrado." << endl;

    return 0;
}
