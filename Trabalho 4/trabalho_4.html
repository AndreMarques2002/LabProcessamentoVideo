Trabalho 4 - Elaboração do Laboratorio Experimental do SPV (Lex)

Roteiro

Bem-vindo ao teste do Sistema de Processamento Visual (SPV) para reconhecimento de placas veiculares. Este roteiro irá guiá-lo passo a passo para realizar o teste de maneira simples e eficiente.

Introdução

O objetivo do teste é verificar se o sistema SPV consegue identificar corretamente as placas veiculares cadastradas no banco de dados e rejeitar as que não estão registradas.

Preparação

Antes de começar, certifique-se de que o Tesseract está instalado na máquina. Caso esteja executando o programa pela primeira vez, abra o terminal e execute os comandos abaixo: 


$ sudo apt-get update
$ sudo apt-get install libsdl2-ttf-dev

Etapas do Teste

Configuração Inicial

Abra a pasta placas_opencv no explorador de arquivo e, depois de aberta, abra o terminal e digite os comandos abaixo: 


$ sudo apt install libtesseract-dev 
$ cmake .
$ g++ -o placas_opencv placas_opencv.cpp `pkg-config --cflags --libs opencv4` -ltesseract `sdl2-config --cflags --libs` -lSDL2_ttf
$ ./placas_opencv

Reconhecimento de Placas Cadastradas

Após inicializar o programa, atente-se aos comandos. Aperte as teclas q para interromper a execução e Espaço para capturar a placa.

Em seguida, selecione uma das placas disponibilizadas e aponte-a para a webcam (é importante enquadrar a placa adequadamente). Ao todo, há 4 placas, sendo que uma delas não está registrada no banco de dados:

Placa Motorista Marca do veículo Modelo do veículo
BRA2E19 Andre Ford Fiesta
ABC1D34 Gabriel Chevrolet Onix
PBA2019 Vinicius Honda Fit
RIO2A18 - - -

Avalie placa a placa, e verifique se a leitura foi realizada corretamente. Deverá aparecer a seguinte mensagem: 


Acesso permitido!
Motorista: Driver X
Modelo do carro: Car X

Coleta de Resultados

Anote os seguintes dados para cada teste:

  • Texto detectado pelo sistema

  • Tempo de detecção (aproximadamente, em segundos)

  • Precisão (se o texto detectado corresponde ao esperado)

Finalização do Teste

Após realizar todos os testes, pressione a tecla q para sair do programa. Responda ao questionário fornecido para registrar suas impressões sobre o sistema.

Questionário final

Ao final do laboratório, responda às perguntas abaixo:

  • O sistema foi capaz de detectar as placas corretamente?

  • Você encontrou problemas ao operar o sistema? Descreva.

  • Como você avalia a precisão do sistema?
